
Neuro Survey — Full Stack (Node + Express + MongoDB + HTML/CSS/JS)

Structure:
- server/: Node.js + Express backend (MongoDB via Mongoose)
- client/: Static frontend (HTML/CSS/JS)

Important:
- Replace MONGODB_URI in .env with your MongoDB connection string (use MongoDB Atlas or local).
- Run server: cd server && npm install && npm run dev
- Open client: open client/index.html in browser (or serve via static server).
- For local testing the backend runs on http://localhost:4000
